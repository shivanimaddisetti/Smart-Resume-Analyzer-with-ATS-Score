# Smart Resume Analyzer with ATS Score

## Overview
The Smart Resume Analyzer is an AI-powered web application designed to help job seekers optimize their resumes for Applicant Tracking Systems (ATS). 
The system analyzes uploaded resumes, compares them with job descriptions, and generates an ATS compatibility score along with personalized improvement suggestions.

## Features
- Upload resumes in PDF format.
- Extract and analyze resume content.
- Calculate ATS score based on keyword matching.
- Compare resumes with job descriptions.
- Identify missing keywords and skills.
- Provide recommendations to improve resume quality.
- User-friendly web interface.

## Project Workflow
1. User uploads a resume.
2. The system extracts text from the resume.
3. Resume content is compared with the job description.
4. ATS score is generated based on keyword relevance.
5. Suggestions and missing keywords are displayed to the user.
